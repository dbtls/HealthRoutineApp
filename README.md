# AI 운동루틴 추천 앱 Seven T


## 소개
- 개인의 체성분, 환경, 신체 상황 등을 기반으로 맞춤형 운동 루틴을 추천해 주는 애플리케이션
- 텍스트 입력 형식의 챗봇을 GUI의 형태로 바꿔 보다 직관적이고 쉽게 운동 설정 가능
